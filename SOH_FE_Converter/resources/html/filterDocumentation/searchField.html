Search field

The documentation for the searchField, describes how to fill it in classic or complex mode.

Classic mode

You can fill the search field with a simple string to search for an occurence into the table for the fields : Galaxy Name, Sector name, System Name, Planet Name

Complex mode

With this mode, you can launch complex research with multiple search key at the same time.
The field will detect automatically the use of a complex expression, and a warning popup will show up if there are some errors with your expression

A complex research is composed of one or more expression which are used to filter the table

expression1 ; expression2

; is the separator required between two search filter

  • expression1 - First expression, for example to filter only the sector which contained "baleck" string
  • expression2 - another independant expression which filter with another field !
